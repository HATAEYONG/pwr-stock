## 현재 vs 목표

| 항목 | 현재 (AS-IS) | 목표 (TO-BE) |
|------|-------------|-------------|
| Feature 종류 | OHLCV + 기본 지표(MA/RSI/MACD/BB) ~20개 | 거래량 세그먼트·섹터 모멘텀·뉴스센티먼트·시장국면 추가 ~50개+ |
| 실제 수익 검증 | 백테스팅 결과에 ML 신호 승률 연동 없음 | ML 신호→백테스팅 파이프라인 직결, 실측 승률·MDD 자동 출력 |
| 모델 갱신 | 수동 재학습(CLI 실행) | 매주 자동 재학습 (Celery Beat 스케줄) |
| 리스크 지표 | MDD 계산 있으나 ML 신호 필터에 미반영 | MDD 기반 포지션 사이징 로직 추가 |
| 아키텍처 | 단일 Predictor 클래스 (1073줄) | Feature Store + Model Registry + Backtesting Bridge 분리 |
